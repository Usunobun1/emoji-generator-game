## Features
- Generates random **cute emojis**:hearts, animals, sparkles, sweets, and more  
- **Pop animation** when a new emoji appears  
- **Copy Emoji** button to save your favorite emoji  
- Fully **responsive** for mobile and desktop  
- Pastel pink theme and quirky font for a fun, playful feel  

## How to Use
1. Open the page in your browser  
2. Click **Generate Emoji** to get a random cute emoji  
3. Click **Copy Emoji** to copy it to your clipboard
